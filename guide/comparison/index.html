# Comparison

When people migrate, they would like to know what to expect from the scoring. So here is a comparison between the original zxcvbn (opens new window) and rewritten zxcvbn-ts (opens new window)

The scoring could have slight changes' duo to various reasons.

  • updated dictionaries (generally more words and dictionaries that are up-to-date)
  • bug fixes
  • more keyboard layouts

If you think the scoring is completely out of place, please open a discussion (opens new window) inside the repository.
